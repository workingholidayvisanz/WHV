WHV Scripts — NZ Immigration

Política de Privacidade

Última atualização: Julho de 2025  ·  Versão 1.0

Esta política descreve como a extensão WHV Scripts coleta, usa e protege os dados dos usuários. Leia com atenção antes de utilizar o produto.

1. O que é o WHV Scripts

O WHV Scripts é uma extensão do Google Chrome que automatiza o preenchimento do formulário de Working Holiday Visa (Visto de Férias-Trabalho) da Nova Zelândia, disponível em onlineservices.immigration.govt.nz.

O produto funciona com um sistema de licença: o usuário adquire uma chave de acesso, digita na extensão, e seus dados são carregados automaticamente para preencher o formulário.

2. Dados coletados

Para funcionar, a extensão coleta e processa os seguintes dados fornecidos voluntariamente pelo usuário por meio do formulário de cadastro:

  • Dados de identificação pessoal: nome completo, data de nascimento, gênero, nacionalidade, endereço atual
  • Documento de viagem: número do passaporte, data de expiração
  • Informações de saúde: respostas às perguntas médicas exigidas pelo formulário WHV (tuberculose, condições cardíacas, etc.) — essas respostas são fornecidas pelo próprio usuário
  • Dados do visto atual (quando aplicável): data de chegada à Nova Zelândia, tipo e validade do visto
  • E-mail de contato: utilizado exclusivamente para envio da chave de licença

3. Como os dados são usados

Os dados coletados são utilizados exclusivamente para:

  • Preencher automaticamente o formulário de WHV no site oficial da NZ Immigration
  • Validar a chave de licença do usuário e verificar sua validade
  • Enviar a chave de licença por e-mail após confirmação do pagamento

Os dados não são utilizados para fins de marketing, publicidade, análise comportamental ou qualquer outro propósito não descrito nesta política.

4. Armazenamento e segurança

Após a ativação da licença, os dados do usuário são salvos localmente no dispositivo, no armazenamento interno do Chrome (chrome.storage.local). Esses dados ficam acessíveis apenas pela extensão instalada no próprio computador do usuário.

Os dados também ficam armazenados em um sistema seguro do desenvolvedor (Google Sheets e Google Drive), com acesso restrito ao titular do produto, exclusivamente para fins de suporte e validação de licença.

Não utilizamos cookies, rastreadores ou qualquer tecnologia de monitoramento.

5. Compartilhamento com terceiros

Os dados do usuário não são vendidos, alugados ou compartilhados com terceiros.

A extensão realiza chamadas para um serviço próprio do desenvolvedor (Google Apps Script), utilizado exclusivamente para validar a licença e recuperar os dados do usuário. Esse serviço é controlado e operado pelo próprio desenvolvedor do WHV Scripts, não por terceiros.

Os dados de preenchimento do formulário são enviados diretamente ao site onlineservices.immigration.govt.nz (governo da Nova Zelândia), conforme a ação do próprio usuário ao submeter a aplicação de visto.

6. Retenção dos dados

Os dados ficam armazenados localmente no Chrome até que o usuário remova a extensão ou limpe os dados de armazenamento manualmente. No sistema do desenvolvedor, os dados são mantidos pelo prazo de validade da licença e por até 12 meses após o encerramento, para fins de suporte.

O usuário pode solicitar a exclusão dos seus dados a qualquer momento pelo canal de contato abaixo.

7. Direitos do usuário

O usuário tem direito a:

  • Solicitar acesso aos dados armazenados
  • Solicitar a correção de informações incorretas
  • Solicitar a exclusão completa dos dados
  • Revogar o consentimento de uso a qualquer momento

Para exercer qualquer um desses direitos, entre em contato pelo e-mail abaixo.

8. Menores de idade

O WHV Scripts não é destinado a menores de 18 anos. Não coletamos conscientemente dados de menores. Caso identifiquemos que dados de um menor foram fornecidos, os excluiremos imediatamente.

9. Alterações nesta política

Esta política pode ser atualizada periodicamente. Alterações significativas serão comunicadas por e-mail. A data da última atualização está sempre indicada no topo desta página.
